# Rainmeter Skins Collection

A personal collection of Rainmeter skins, including both custom creations and curated third‑party skins.

## 📂 Skins

- **MiniWeatherSimple** – Lightweight weather widget using Open‑Meteo + PowerShell. Auto‑geocoding, icon scale, monochrome toggle.
- **Spectrum** – Visual audio spectrum analyzer.
- **Mond** – Elegant clock and recycle bin (third‑party, included for convenience).
- **lines** – System monitoring lines (CPU, RAM, HDD) – minimal design.
- **EdgeDock** – Application dock with custom icons.

Other folders (`@Vault`) contain plugins and shared resources.

## 🚀 Usage

1. Copy the desired skin folders into your Rainmeter `Skins` directory (usually `Documents\Rainmeter\Skins`).
2. Load skins via Rainmeter’s Manage window.
3. Some skins (e.g., MiniWeatherSimple) may require PowerShell execution policy adjustment – see individual skin folders for details.

## 📝 Notes

- Skins are provided as‑is; some are my own work, others are adapted or kept for personal use.
- Icon sets, fonts, and plugins are included where needed.
